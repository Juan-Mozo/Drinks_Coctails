# Drinks & Cocktails

Drinks & Coctails is a simple Android App that uses a Room database pre-populated with a external SQL database, which contains 16 cocktail recipes in english and spanish.

## ARCHITECTURE
- MVVM
- Clean Architecture

## LIBRARIES
- Room
- Jetpack Compose
- Dagger-Hilt
- Coroutines
